# LAB08 – Iteration, Context Managers, Descriptors

## Description
This project demonstrates how Python protocols define behavior:
- iteration protocol
- context manager protocol
- descriptor protocol

## Setup

python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

## Run

python src/main.py

## Type checking

mypy --strict .

## Notes

The program shows:
- how objects work with for loops
- how with statement controls execution
- how descriptors validate attributes
